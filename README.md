# Task 1: Data Immersion & Wrangling

## Internship
ApexPlanet Remote Internship

## Objective
The objective of this task is to understand the dataset, assess data quality, and clean and transform the data to make it analysis-ready.

## Dataset Overview
The dataset contains customer demographic and sales-related data including age, gender, country, signup date, last purchase date, and total purchase amount.

## Data Quality Issues Identified
- Missing values in age and last_purchase_date
- Duplicate customer records
- Inconsistent date formats

## Data Cleaning & Transformation
- Removed duplicate records
- Filled missing values using median and default values
- Standardized date formats
- Created a new feature: customer_tenure_years

## Tools Used
- Python
- Pandas
- VS Code
- GitHub

## Final Output
The cleaned dataset is saved as cleaned_customer_dataset.csv and is ready for analysis.

## Conclusion
This task provided hands-on experience with real-world data wrangling and cleaning techniques.
